About Starshooter

Starshooter is a game about shooting stuff in space. It shoots automatically and the player controls the aiming point with a phone sticked to a wheelchair. The goal is to shoot as many planets as possible until the background music stops. It's pretty cool : every time you shoot is pretty cool and there is always a sense of urgency. I give this game 10/10 good game.

Motivations

The main theme of the AGI project 2 is to create a game that could be useful for improving the daily life of some people. So we have decided to create a game to help to learn how to use a wheelchair. Especially, this game helps to learn how to tilt correctly with the wheelchair.

Furthermore, another motivation is to learn to code with the WebGL/Three.js library. During our first projects in Advanced Graphics and Interaction, we used Unity, a game engine, to develop our games. So for this project, we want to come back to classic computer programming.

Technology

We use a wide range of technology including computers and monitors. There is always a wheelchair involved aswell.

Challenges

Lessons learned

- Three.js is a good library to use in computer graphics but it is quite complicated to learn to use it and to debug it because the documentation is not really detailed
- Three.js examples found on the web generally can be difficult to use because some of them don't use the latest version of Three.js so the code didn't work anymore if you want to use the latest version since some functions have been replaced.
